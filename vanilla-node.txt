node app.js

This will set and run the server on port 5000.

R:  Go over to Postman and SEND a GET request. The request URL as http://localhost:5000/api/todos

R:  Go over to Postman and SEND a GET request. Enter the request URL as http://localhost:5000/api/todos/:id, where :id is the id of the single todo you want to fetch


D:  Go over to Postman and SEND a DELETE request. Enter the request URL as http://localhost:5000/api/todos/:id, where :id is the id of the single todo you want to delete,

U:  Go over to Postman and SEND a PATCH request. Enter the request URL as http://localhost:5000/api/todos/:id, where :id is the id of the single todo you want to update


C:  Go over to Postman and open a new tab, select a POST request and enter the request URL as http://localhost:5000/api/todos.
